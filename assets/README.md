# Assets Folder

Place your images and files in this folder:

## Required Assets:

1. **profile.jpg** - Your profile photo
   - Recommended size: 300x300px or larger
   - Format: JPG or PNG
   - Should be a professional headshot

2. **CV.pdf** - Your curriculum vitae
   - Format: PDF
   - Keep file size reasonable (under 5MB)

3. **Publication Images** (publication1.jpg, publication2.jpg, etc.)
   - Recommended size: 250px width
   - Format: JPG or PNG
   - Can be paper figures, diagrams, or screenshots

4. **Organization Logos** (logo1.png, logo2.png, etc.)
   - Recommended size: 120px max height
   - Format: PNG (with transparent background preferred)
   - Logos of companies/institutions from your experience

## Optional:

- Add a favicon.ico for your website icon
- Add og-image.jpg for social media sharing preview

## Tips:

- Optimize images before uploading (use tools like TinyPNG)
- Use descriptive filenames
- Keep file sizes small for faster loading
- Use consistent image sizes for a professional look
